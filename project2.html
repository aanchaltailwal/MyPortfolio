Lost and Found Web application

A web application for the university to streamline the retrieval of lost items. Losing valuable items is a significant issue, and having access to a list of recently found items can be incredibly helpful. If your lost item appears on the list, it can save you considerable time and effort.

Technical Skills : MongoDB, ReactJS, NodeJS, ExpressJs, Netlify, Cyclic

DevOps Tools : Jira, GitHub, Jenkins, Nexus
